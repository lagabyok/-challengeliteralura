# Tienda Aurelion

### Análisis de Rendimiento de Ventas (Demo IBM SkillsBuild)

Este repositorio contiene el Análisis Exploratorio de Datos (EDA) del rendimiento de ventas para **Tienda Aurelion**, una distribuidora de productos de consumo masivo (Alimentos y Limpieza). El objetivo es analizar los datos del primer semestre para extraer *insights* accionables, identificar patrones de consumo y validar hipótesis de negocio.
